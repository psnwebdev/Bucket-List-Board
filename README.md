# 🪣 Bucket List Board

A beautiful, fully functional bucket list web app built with plain HTML, CSS, and JavaScript. No frameworks, no dependencies — just open and use.

---
## Live Demo :
https://psnwebdev.github.io/Bucket-List-Board/

## 📁 File Structure

```
bucket-list/
├── index.html      # Page structure and markup
├── styles.css      # All styling and animations
├── script.js       # All logic and interactivity
```

---

## ✨ Features

- **Add items** — Enter a title, pick a category and priority, add an optional note, then press Enter or click "Add to List"
- **Edit items** — Click the ✏️ pencil icon on any card to edit title, note, category, and priority inline
- **Delete items** — Click the 🗑 trash icon to remove a card with a smooth fade-out animation
- **Mark as done** — Click the ✓ checkmark to toggle completion; done items get a strikethrough and green tint
- **Filter** — Filter by All / To Do / Done / or any category using the pill buttons
- **Stats bar** — Live count of total, completed, and remaining items
- **Persistent storage** — All data is saved to `localStorage`, so your list survives page refreshes and browser restarts
